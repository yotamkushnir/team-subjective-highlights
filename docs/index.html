How winner-channel vs loser-channel edits differ (club-blind)

This view compares the winning club’s official highlight to the losing club’s for the same match. All headline numbers are role-based (winner edit / loser edit), not brand-based. We do not try to explain which PBP events were included—goals and obvious top plays are given. The signal we model is pacing, volume splits, persona/emotional time, replay counts, and optional rival exposure.
